This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.1.4.


## Prepare Project
1. Copy `.env.copy` to `.env`
2. Get `API_KEY` in [Weather Api](https://www.weatherapi.com/)
3. Change `API_KEY` in `.env`

## Development server
```bash
npm run build:ssr
npm run serve:ssr
```
